# Gamehub

Веб-приложение для скачивания файлов. Позволяет просматривать и скачивать файлы для разных игр через браузер.

## Установка

```bash
pip install -r requirements.txt
python app.py
```

## Добавление файлов

Положите файлы в папки:
- `static/downloads/ddnet/`
- `static/downloads/cs2/`
- `static/downloads/minecraft/`
